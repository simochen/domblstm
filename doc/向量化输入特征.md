###1. 生成单个序列fasta文件、ssa特征、目标序列文件
	getSeq.pl domfile dir
	
	./genFeature/getSeq.pl ./dataset/multi354.dom /home/chenxiao/dataset/multi

###2.1 运行blastpgp，
	runBlast.pl dir
	
	./genFeature/runBlast.pl /home/chenxiao/dataset/multi
###2.2 批量运行blastpgp
	runBlast_batch.pl dir start end
	
	./genFeature/runBlast_batch.pl /home/chenxiao/domainDS/cullpdb/multi 1 720

###3. 将全部序列txt文件，分割成10个一组的片段fasta文件
	getDomSeg.pl _dom.txt dir
	
	./getFeature/getDomSeg.pl ./dataset/multi354_dom.txt /home/chenxiao/dataset/multiSSA

###4.1 对每个片段fasta文件运行 SRATCH 提取特征
	runScratch.pl dir
	
	./getFeature/runScratch.pl ./dataset/multiSSA
###4.2 对片段fasta文件批量运行 SCRATCH 提取特征
	runScratch_batch.pl dir start end
	
	./getFeature/runScratch_batch.pl ./domainDS/cullpdb/multiSeg 1 72

###5. 从 .ss .ss8 .acc .acc20 中生成特征
	getSSA.pl sstype[ss,ss8] satype[acc,acc20] segNum indir (outdir)
(or getSSAalo.pl)

	./getSSA.pl ss8 acc20 36 ./dataset/multiSSA ./dataset/multi_s8a20

###6. 生成PSSM特征，合并生成输入特征
	getFeat.pl pssmdir ssadir ssatype seqNum (outdir)
	
	./genFeature/getFeat.pl /home/chenxiao/dataset/multi /home/chenxiao/dataset/multi_s8a20 ss8acc20 354

###7. 批量运行 DISOPRED 提取特征
	runDiso.pl dir start end
	
	./genFeature/runDiso.pl ./domainDS/cullpdb/multi_diso 1 720

###8. 批量生成 DISO 特征
	getDISO_batch.pl dir start end

	./genFeature/getDISO_batch.pl ./domainDS/cullpdb/multi_diso 1 720

###9. 与pssm、ssa特征合并生成25维输入特征
	getFeat+diso.pl ssadir disodir segNum (outdir)
	
	./genFeature/getFeat+diso.pl ./domainDS/cullpdb/multi_sa ./domainDS/cullpdb/multi_diso 720 ./domainDS/cullpdb/multi_f25
